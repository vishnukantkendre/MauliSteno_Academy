Programming Fundamentals

Duration - 4 Weeks

This module is designed for individuals who are new to programming and have no prior experience in coding and ideal for people from non-cs background. It will provide an overview of programming fundamentals and introduce participants to programming concepts and techniques.

Topics Covered
  • Introduction To Programming Concepts
  • Programming Languages and Paradigms
  • Java Language Basics
  • Variables And Data Types
  • Operators And Expressions
  • Control Structures (Conditional Statements and Loops)
  • Functions And Classes
  • Debugging And Error Handling
  • Array Basics
  • Introduction To Strings
Module Outcome

This module will give you complete understanding of programming fundamentals, concepts & techniques and by end of this module you will be ready to write code in any langauge.

Introduction to Data Structures & Algorithms

Duration - 4 Weeks

If your goal is to land a job at a top product company, proficiency in data structures and algorithms, as well as problem-solving skills, are essential. This module aims to provide you with a solid foundation in data structures and algorithms, equipping you with the knowledge and skills needed to excel in online assessments and interviews with leading tech companies.

Topics Covered
  • Time & Space Complexity Analysis & Problem solving
  • Intro & Problem Solving In Arrays
  • Array Memeory Management
  • 2-D / N-D Arrays
  • Array Operations (Transformation / Rotation)
  • Two Pointers
  • Prefix & Suffix Array
  • Sub Array Problems
  • Strings
  • Pattern Matching Problems
  • Pattern Searching Problems
  • Substring Problems
  • Search Algorithms
  • Sorting Algorithms
  • LinkedList
Module Outcome

This module will give you strong foundation for problem solving using Data Structures & Algorithms. By end of this module you will be ready to solve easy and medium level questions using Arrays, Strings and LinkedList and ready for more advanced Data Structures and problem solving.
